# SiS Model B

**Siegel in Silico (SiS)** — an AI-powered recipe creativity assistant with a Python/FastAPI backend and React/Vite frontend.

## Changelog

### v0.1.0 — Initial Release (2026-03-27)

- Project scaffolded with Python/FastAPI backend and React/Vite frontend
- Multi-provider LLM support (Anthropic, OpenAI, Google)
- Single orchestrator agent with tool-based sub-agents
- Kitchen/pantry management system
- Real-time SSE streaming chat interface
- Recipe creation, scaling, and saving
- Onboarding flow (chef / home cook modes)
